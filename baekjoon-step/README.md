# 백준 단계별 문제 풀이 (JavaScript)

백준의 단계별 문제를 JavaScript(Node.js)로 풀이한 저장소입니다.

단계별 문제 링크: [https://www.acmicpc.net/step](https://www.acmicpc.net/step)

## 목표

- 단계별 문제를 꾸준히 풀이
- 기본 문법과 문제 해결력 강화
- 풀이 기록을 날짜/문제번호 기준으로 관리

- [`5단계_문자열`](./5단계_문자열/README.md)
- [`6단계_심화_1`](./6단계_심화_1/README.md)

- [`9단계_배수와_약수`](./9단계_배수와_약수/README.md)
- [`14단계_집합과_맵`](./14단계_집합과_맵/README.md)

## 파일 규칙

- 파일명 형식: `YYYYMMDD_문제번호.js`
- 예시: `20260329_2480.js`

## 실행 방법

1. 테스트 입력을 `input.txt`에 작성합니다.
2. 아래 명령어로 실행합니다.

   ```bash
   node 2단계_조건문/20260329_2480.js < 2단계_조건문/input.txt
   ```
