# 8단계 수학 1 학습 메모

수학 1 단계를 풀면서 자주 쓰는 문법과 핵심 개념을 정리합니다.

## 1) B진법 문자열을 10진법으로 (`2745번`)

- B진법 문자열 N은 오른쪽부터 각 자리값에 𝐵^𝑖를 곱해 모두 더하면 10진수로 변환된다. 예를 들어 10진수라면 자리마다 10의 거듭제곱을 곱해 더한 값으로 `123 = 1×10^2 + 2×10^1 + 3×10^0`이다.
  - 진법에 따라 사용할 수 있는 숫자가 다르다.
    - 2진수: 0~1
    - 8진수: 0~7
    - 16진수: 0\~9, A\~F
    - 36진수: 0\~9, A\~Z
- 한 문자를 B진법에서 쓰는 수치(0~B-1)로 바꿀 때:
  - `'0' ~ '9'` → `Number(char)` (숫자 그대로 0~9)
  - `'A' ~ 'Z'` → `char.charCodeAt(0) - 'A'.charCodeAt(0) + 10` (A=10, B=11, …, Z=35, 10~35)

```js
const base = Number(B);
let result = 0;
for (let i = 0; i < N.length; i++) {
  const char = N[N.length - i - 1];
  const value =
    char >= "0" && char <= "9"
      ? Number(char)
      : char.charCodeAt(0) - "A".charCodeAt(0) + 10;
  result += value * base ** i;
}
console.log(result);
```


## 2) 벌집 링(층) 구하기 (`2292번`)

- 벌집은 1번을 중심으로 바깥으로 갈수록 층이 확장되며, 각 층의 **최대값(scopeMax)**만 알면 범위를 판단할 수 있다.
- 층이 증가할 때마다 방의 개수는 6, 12, 18, ... (6 × layer)씩 늘어나므로, scopeMax += 6 \* layer로 범위를 확장한다.
- N > scopeMax인 동안 계속 확장하고, N이 포함되는 순간의 layer가 정답이다.
- ps. 우선 표로 그려서 차이 보고, 조건에 맞을 때까지 반복이 필요하다면 **while**을 사용한다. 이 때 **누적 값**이 필요하다면 이를 이용한다.

| layer | 구간 (`scopeMin`~`scopeMax`) | 이번 링에서 더한 칸 수 |
| ----: | ---------------------------- | ---------------------- |
|     1 | 1                            | (시작)                 |
|     2 | 2~7                          | +6                     |
|     3 | 8~19                         | +12                    |
|     4 | 20~37                        | +18                    |

```js
let layer = 1;
let scopeMax = 1;
while (N > scopeMax) {
  scopeMax += 6 * layer;
  layer++;
}
console.log(layer);
```

## 3) 대각선 분수에서 X번째 구하기 (`1193번`)

- 분수들을 줄 단위로 보면, k번째 줄에는 k개의 분수가 있고, 각 줄의 분자와 분모의 합은 항상 `k + 1`로 일정하다.
- X가 최대 1천만까지 갈 수 있으므로, 모든 분수를 배열에 쌓지 말고 **줄 번호와 offset만으로 분자·분모를 직접 계산**해야 메모리·시간 모두 안전하다.
  - `line`는 줄 번호, `max`는 그 줄의 마지막 위치로, X가 속한 줄은 마지막 위치(누적 인덱스 `max`)로 찾는다.
  - `offset = max - X`로 같은 줄 끝에서부터 몇 칸인지 잡은 다음 **짝수 줄**(`line % 2 === 0`)은 분모가 커지는 방향(예: `1/4`, `2/3`, `3/2`, `4/1`), **홀수 줄**은 반대 방향(예: `3/1`, `2/2`, `1/3`)으로 계산한다.

|  줄 | 구간(누적 끝 `max`) | 예시(해당 줄의 분수들)     |
| --: | ------------------- | -------------------------- |
|   1 | 1                   | `1/1`                      |
|   2 | 3                   | `1/2`, `2/1`               |
|   3 | 6                   | `3/1`, `2/2`, `1/3`        |
|   4 | 10                  | `1/4`, `2/3`, `3/2`, `4/1` |

```js
let max = 0;
let line = 0;
while (max < X) {
  line++;
  max += line;
}
const offset = max - X;
const isEvenNumber = line % 2 === 0;
let numerator, denominator;
if (isEvenNumber) {
  numerator = line - offset;
  denominator = offset + 1;
} else {
  numerator = offset + 1;
  denominator = line - offset;
}
console.log(`${numerator}/${denominator}`);
```

## 4) 달팽이 올라가기 — 일 수를 수식으로 (`2869번`)

- 낮에 `A`만큼 올라가고 밤에 `B`만큼 미끄러진다. 마지막 날에는 정상에 도달하면 더 이상 미끄러지지 않는다.
- 따라서 마지막 날을 제외한 하루 순수 상승량은 `A - B`이고, 정상 직전 높이는 최소 `V - B` 이상이어야 한다.
- 필요한 총 일수는 다음 수식으로 계산할 수 있다: `(V - B) / (A - B)`
- `V`가 최대 10억까지 올 수 있으므로, 하루씩 `+A`, `-B`를 반복하는 시뮬레이션은 반복 횟수가 너무 커져 시간 초과가 발생한다. **수식 한 번**으로 계산해야 안전하다.

```js
const days = Math.ceil((V - B) / (A - B));
console.log(days);
```
