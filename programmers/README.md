# 프로그래머스 문제 풀이 (JavaScript)

프로그래머스 문제를 JavaScript(Node.js)로 풀이한 저장소입니다.

## 목표

- 문제를 꾸준히 풀이
- 기본 문법과 문제 해결력 강화
- 풀이 기록을 날짜/문제번호 기준으로 관리

## 핵심 내용 기록

- [`learn/challenges/lv1`](./learn/challenges/lv1/README.md)

## 작성 형식

- 파일명 형식: `YYYYMMDD_문제번호.js`
- 파일명 예시: `20260422_12950.js`

## 실행 방법

1. 테스트할 입력값을 직접 설정해 실행합니다.
2. 아래 명령어로 실행합니다.

   ```bash
   node programmers/learn/challenges/lv1/20260422_12950.js
   ```
